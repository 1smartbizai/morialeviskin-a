
This directory contains screenshots of the Bellevo app interface used in the marketing landing page.

Screenshots should be named according to the screen they represent:
1. dashboard.png - Dashboard main view
2. analytics.png - Analytics and reporting screen
3. clients.png - Client management view
4. calendar.png - Calendar and appointments view
5. payments.png - Payments and financial management

Screenshots should be high quality images (preferably 1080px wide or larger) with a 16:9 or similar mobile-friendly aspect ratio.

To capture these screenshots:
1. Login to the admin dashboard
2. Navigate to each page
3. Use browser developer tools to set a mobile viewport size
4. Take a clean screenshot of just the main content
5. Save in PNG format in this directory
